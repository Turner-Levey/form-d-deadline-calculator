# Form D Deadline Calculator

Free no-tracking Form D deadline calculator for Regulation D first-sale timing:

- 15-calendar-day notice target from the first-sale date.
- Weekend, standard U.S. federal holiday, and optional custom closure handling.
- Actual filing-date status comparison.
- Annual amendment anniversary and planning target for ongoing offerings.
- EDGAR access note, source-linked memo copy, and CSV export.

The tool runs entirely in the browser. It has no signup, cookies, analytics, beacons, external scripts, uploads, or browser storage.

Live site: https://form-d-deadline-calculator.vercel.app/
